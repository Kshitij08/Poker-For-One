# Poker for One - Main Game

A single-player poker game where you play against AI in a best-of-3 match.

## Game Rules

1. Each player starts with 7 cards
2. Each player can discard up to 2 cards (optional)
3. After discarding, new cards are drawn from the deck
4. Each player selects 5 cards to play
5. Best poker hand wins the round
6. First to win 2 rounds wins the game

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

The game will open at http://localhost:3000

## Build

```bash
npm run build
```
